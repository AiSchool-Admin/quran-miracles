## نظام المستويات للارتباطات العلمية

كل ارتباط يُقيَّم بـ 7 معايير (0-10):

1. وضوح لغوي
2. استقلال تاريخي (هل كانت المعرفة موجودة قبل الإسلام؟)
3. دعم تفسيري قبل-حديث
4. خصوصية
5. قابلية الدحض
6. إجماع ترجمي
7. تماسك سياقي

| المجموع | المستوى |
|---------|---------|
| ≥ 60/70 | tier_1 |
| 45-59/70 | tier_2 |
| < 45/70 | tier_3 |

## نظام المستويات للفرضيات التنبؤية

| المستوى | الحالة | الشرط |
|---------|--------|-------|
| tier_0 | نمط أولي — داخلي فقط | لا يوجد p-value بعد |
| tier_1 | فرضية مبدئية — مع تحذير | p < 0.05 + FDR + Effect size > 0.3 |
| tier_2 | ارتباط محتمل — مع شارة | p < 0.01 + مراجعة خبير |
| tier_3 | نتيجة موثقة — تفاصيل كاملة | p < 0.001 + Bayes > 10 |
| tier_4 | اكتشاف مؤكد — مرجع أكاديمي | نشر محكّم + إجماع أكاديمي |
